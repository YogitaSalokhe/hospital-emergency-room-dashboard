# hospital-emergency-room-dashboard
Hospital Emergency Room Analysis Dashboard built using Excel to analyze patient flow, wait times, and hospital service efficiency.

## Project Overview
This project analyzes Hospital Emergency Room data to help stakeholders monitor patient flow, waiting time, and hospital service efficiency.

The dashboard provides key insights that help hospital management improve patient experience and optimize emergency department operations.

## Objectives
- Monitor number of patients visiting the emergency room
- Track average waiting time
- Analyze patient satisfaction score
- Understand admission vs non-admission rates
- Identify patient demographics
- Track department referrals

## Tools Used
- Excel
- Power Query

## Dashboard Features
- Interactive year filter
- Monthly patient tracking
- Age group analysis
- Gender distribution
- Department referral analysis
- Admission status analysis
